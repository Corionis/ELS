Hint Keys

Hint Keys control Hint processing and which ELS instances are involved. These may include publishers and subscribers.

A Hint Key is the "key" element from a collection JSON file.

UUID List Button

The UUID List button present a list of known collection files and displays the "description" and "key" elements from each file found in the ELS ./libraries/ directory.

Note:

This option must be defined on the command line with the

-k | --keys or -K | --keys-only [file]
option or in an Operation for them to be used. These Navigator dialogs only edit the files.
