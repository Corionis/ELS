Hint Keys

Hint Keys control Hint processing and which ELS instances are involved. These may include publishers and subscribers.

A Hint Key is the "Key" element from a collection JSON file.

Key List Button

The Key List button presents a list of existing Library files and displays the "Description" and "Key" elements from each file found in the ELS ./libraries/ directory.


Note:

This option must be defined on the command line with one of the following options or in an Operation for them to be used in a Job.

-k | --keys [file] or -K | --keys-only [file]

The -K | --keys-only option executes Hints and other back-up options, such as --duplicates, but does not perform back-up operations.
