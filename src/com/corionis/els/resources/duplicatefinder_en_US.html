Duplicate Finder

This is a manually-executed tool that may be run on an ELS Collection. System folders are not supported. Works locally or remotely.

Because ELS and modern media systems span multiple storage devices a duplicate is currently defined as the same filename inside the same library directory name. For example: 1922 Nosferatu on two different sources of the same Library. It is not a filename-level comparison. Other duplicate comparison algorithms may be added in future versions.

This tool will not appear in the Jobs list of available tools.

Cross-check

Normally duplicates are only checked within each library. With cross-check enabled items are checked across all libraries.

Run button

Runs the Duplicate Finder on the selected Browser collection, publisher or subscriber.

Go button

The Go button navigates to the item in the Browser.
