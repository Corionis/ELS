Email Templates

Built-in templates provide a way to invite a user to connect to ELS with information and instructions on getting started depending on the type of user, Basic, Advanced or Administrator.


To invite a user go to the Libraries tab, select the user's library, then go to the User tab and click the Invite button. It has it's own Help documentation.


There are templates for each user type for both HTML and text email formats.


The default templates are built-in. If modified the template is then stored on disk in the .els/local/Templates/ directory. Custom templates are also supported. Use the Default button to restore a default template.


Formatting HTML

Basic HTML is supported. The buttons at the top of the template editor provide some formatting options. Not all HTML tags are provided but may be entered.


Use the Preview and Browser buttons to view the template formatted with substitutions for variables.


When used from System, Email Templates the variables use the currently-loaded Publisher and Subscriber. When used from Libaries, User, Invite the currently-loaded Publisher and current-selected Library populate the variables.


Variable Substitutions

The following variables are substituted for Preview, Browser and the Invite Send button.

Variable Description
${PublisherRepo} Publisher repository Description.
${PublisherHost} Publisher Host.
${PublisherName} Publisher User name.
${UserRepo} Selected User repository Description.
${UserName} Selected User repository User name.
${UserType} Selected User repository User type, Basic, Advanced or Administrator.
