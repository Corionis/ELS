Greetings from ${publisherName} at ${publisherRepo} :

Steps to complete the process of connecting to ${publisherRepo} with ELS:

 1. If ELS is not installed download from https://www.elsnavigator.com/, install and run it once so it creates it's directories.

    a. For Windows use the desktop shortcut to start ELS.
    b. For Linux and Mac run ELS-Navigator in your [Home]/.els/ directory.

 2. If ELS is running please close it.

 3. Unpack the attached file into your [Home]/.els/ directory. You may need to "show hidden files" to see it.

    a. Be sure the directory you select contains at least these subdirectories:
        • bin/
        • libraries/
        • local/
        • rt/
        • system/

 4. Start ELS. Goto System, Preferences and make sure "User-level security" is enabled on the General tab.

 5. Go to File, Open Publisher and open your "${userRepo}" library.

 6. Adjust your ELS setup using the Libraries tab to adjust your General options, Locations and Bibliography as needed, see below.

    a. Do not change any Keys on the Libraries, General tab. They are used for login and to control ELS operations.

 7. Go to File, Open Subscriber and connect to "${publisherRepo}_connection". Be sure to select Remote Connection and Host on the right.


Other Notes

 A. The Libraries tab shows the definition of ${userRepo} and ${publisherRepo}. Those tabs are:

    • General defines the basic options for ELS instances.
    • Locations specify minimum disk space for library sources.
    • Bibliography specifies libraries and their source(s).
    • User specifies Resources (ELS instances) and related permissions.

 B. Locations and Bibliography details for ${publisherRepo} are not provided for security reasons. Only connection options are used.


ELS is available here: https://www.elsnavigator.com/

More information is available in the ELS Wiki Documentation at https://github.com/Corionis/ELS/wiki and ELS Discussions at https://github.com/Corionis/ELS/discussions.
