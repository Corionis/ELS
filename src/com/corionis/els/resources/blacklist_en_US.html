Blacklist

The Blacklist is a simple list of IPv4 IP addresses that are not allowed to connect to this instance when it is operating as a publisher or subscriber listener.

The Whitelist, if defined, is evaluated first, then the Blacklist if defined. The check is performed when a incoming connection is made but before the authentication handshake is attempted between the systems. If the check fails the socket is summarily closed and the occurrence is logged.

Note:

This option must be defined on the command line with the

-B | --blacklist [file]
option or in a listener Operation for them to be used. These Navigator dialogs only edit the files.

Examples:

104.16.148.244

104.16.149.244
