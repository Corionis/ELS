Send Invitation / Update Email

To make the process of getting a new user configured and connected to this ELS instance easier this tool uses Email Templates and builds an archive tar or zip to send the user the needed configuration files.


Options

  • Template
    Select the desired email template in either HTML or text format.
  • Add to Authentication Keys
    To allow the user to connect their Library Key must be in the Authentication Keys.
  • Add to Hint Keys
    Add only if the user will be using Hints and a Hint Tracker/Server.
  • Template
    Add the user's Host IP address to the Whitelist connection filtering.
  • Server
    Select the desired Email Server to send the email. For beginning emails be sure it is a server you have access to so you receive their reply with answers to the beginning questions.
  • To
    The user's email address. The default value is taken from the currently-selected Library General tab, Email address.

Buttons

  • Preview
    View a formatted version of the currently-selected template.
  • Browser
    View a formatted version in your default browser. Sometimes a better representation than Preview.
  • Archive
    Create an archive tar or zip only. If the user's General, Flavor is Windows a zip is created, otherwise a tar file.
  • Send
    Create archive tar or zip for attachment, perform any Adds, then send the selected email to the user
