Cleanup Tool

This tool may be applied to:
  • Collection
  • Library
  • Directory
  • File
Note: There is no Dry-Run option for this tool.

Works locally on the Publisher only and requires a previous Task with a Publisher or a -p|-P option in command line.

Named configurations may be executed:
  • Using the Run button on the selection(s) in the Browser
  • Added as a Task in a Job on the defined Origins

Maximum age in days

The maximum number of days old from "now" of the selecting Origin(s). Any files older are deleted.
