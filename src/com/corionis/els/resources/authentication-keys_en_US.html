Authentication Keys

Authentication Keys control what ELS instances are allowed to access this ELS instance when operating as a listener. Note a subscriber is the same as a publisher. Their role as a publisher or subscriber is controlled by which is defined for the -p|-P and -s|-S options.

Authentication Keys are optional with ELS subscriber and publisher listeners, but are required for the Hint Server listener.

If Authentication Keys are not specified the current publisher is used and is the only client that may connect to the listener.

An Authentication Key is the "key" element from a Library JSON file.

UUID List Button

The UUID List button presents a list of known Library files and displays the "description" and "key" elements from each file found in the ELS libraries/ directory.


Note:

This option must be defined on the command line with the

-A | --auth-keys [file]
option or in an Operation for the keys to be used. These Navigator dialogs only edit the files.
