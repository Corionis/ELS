Authentication Keys

Authentication Keys control what ELS instances are allowed to access this ELS listener instance. Note a subscriber is the same as a publisher. Their role as a publisher or subscriber is controlled by which is defined for the -p|-P and -s|-S options.

Authentication Keys may be used with ELS subscriber and publisher listeners, and are required for the Hint Server listener.

If Authentication Keys are not specified the current publisher is used and is the only client that may connect to the subscriber.

An Authentication Key is the "key" element from a collection JSON file.

UUID List Button

The UUID List button presents a list of known collection files and displays the "description" and "key" elements from each file found in the ELS libraries/ directory.

Note:

This option must be defined on the command line with the

-A | --auth-keys [file]
option or in an Operation for them to be used. These Navigator dialogs only edit the files.
