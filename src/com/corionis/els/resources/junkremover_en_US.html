Junk Remover

This tool may be applied to:
  • Collection
  • Library
  • Directory
  • File
Works locally or remotely.

Named configurations may be executed:
  • Using the Run button
  • Added as a Task in a Job

Junk Pattern

Patterns are similar to common command line wildcard patterns.
  • ? = a single character
  • * = zero or more characters

Case?

Enable to make the pattern case-sensitive. Disable for case-insensitive.
