Archiver Tool

This tool may be applied to:
  • Collection
  • Library
  • Directory
  • File
Note: There is no Dry-Run option for this tool.

Works locally on the Publisher only and requires a previous Task with a Publisher or a -p|-P option in command line.

Named configurations may be executed:
  • Using the Run button on the selection(s) in the Browser
  • Added as a Task in a Job on the defined Origins

Append Publisher & Subscriber

Appends the current or last Publisher and Subscriber names to the archive filename.

Append date & time

Appends the current date and time to the archive filename.

Delete files

Remove the files that have been archived.

Target directory & base name

Select and/or enter the directory and optional filename (prefix). Either Append options or a filename are required. Both may be used.

To use a directory only and the generated filename add a separator (/ or \) to the end of the target. Any directories that do not exist will be created.
