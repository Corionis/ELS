Junk Remover

This tool can be applied to:
  • Collection
  • Library
  • Directory

Junk Pattern

Patterns are similar to common command line wildcard patterns.
  • ? = a single character
  • * = zero or more characters

Case?

Enable to make the pattern case-sensitive. Disable for case-insensitive.

Command Line

  • -D | --dry-run
  • -l | --library [name] : Library to process, no name means all libraries
  • -L | --exclude [name] : Library to exclude, name required
  • -p | --publisher : Run on publisher
  • -s | --subscriber : Run on subscriber
