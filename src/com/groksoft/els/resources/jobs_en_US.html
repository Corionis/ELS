Jobs

Jobs execute defined Tasks in the sequence defined. A Task is a defined Tool with starting Origin(s) as arguments to the Tool.

Tasks

The Add button for Tasks displays a list of Tools that have been created previously. The list of Tools is updated each time the button is clicked.

Task Origins

Select the publisher and/or subscriber accordingly by clicking the ... button.
  • "Any publisher" will use any loaded publisher
  • "Specific publisher" allows selection of a publisher from the list of libraries
  • "Any subscriber" will use any loaded subscriber
  • "Specific subscriber" allows section of a subscriber from the list of libraries
The Add button uses whatever is currently selected in the Browser. That means combinations can be assembled that do not necessarily make sense or will execute correctly.

Generate

The Generate button creates a basic command line to execute the currently-selected job. Edit and copy (Ctrl-C) as desired.

Depending on your choices additional parameters may be required for the command line job to run correctly. In particular, if "Any publisher" or "Any subscriber" has been selected for the job then -p and/or -s parameters will needed.
