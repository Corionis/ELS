Getting Started

ELS uses a publisher/subscriber paradigm to present a collection of logical libraries from a local publisher to a subscriber, or back-up. The subscriber may be local, with all storage devices attached, or remotely using another ELS instance operating as a Subscriber Listener.

Each logical library may span multiple storage devices. This is the same approach used by modwen media streaming systems such as Plex but is also useful for anyone producing large volumes of content, videos, music, data, etc. Each storage Location may specify a "Minimum Free Space". A library is defined as a Bibliography of storage Location used by the library.

ELS may be run in a variety of modes. The visible mode is this ELS Navigator. However several other background modes are available to automate ELs operations.

Start-up Steps - Synopsis

  1. Define a publisher on the Libraries tab.
  2. Use File, Open Publisher to open the new publisher.
  3. Define one or more subscribers on the Libraries tab.
  4. Use File, Open Subscriber to open the new subscriber.
  5. Create a Navigator Operation using the Tools, Operations menu option.
    • Use the Generate button to create a Desktop Shortcut to start the ELS Navigator with the open publisher and subscriber.
