Authorization Keys

Authorization Keys control what ELS instances are allowed to access this ELS instance. Those may include publishers and Navigators. Note a subscriber is the same as a publisher. Their role as a publisher or subscriber is controlled by which is defined for the -p|-P and -s|-S options.

An Authorization Key is the "key" element from a collection JSON file.

UUID List Button

The UUID List button present a list of known collection files and displays the "description" and "key" elements from each file found in the ELS ./libraries/ directory.

NOTE: These options must be defined on the command line or in an Operation for them to be used. The Navigator dialogs only edit the files.
